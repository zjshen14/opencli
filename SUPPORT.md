# Support

Welcome to the OpenCLI community! We're thrilled to have you here. If you need help, there are several ways to get support.

## Getting Help

### 1. GitHub Issues
If you have encountered a bug or have a specific feature request, please open an issue in this repository. 
- [Report a Bug](../../issues/new?template=bug_report.md)
- [Request a Feature](../../issues/new?template=feature_request.md)

*Note: Please search existing issues to see if your problem has already been reported before creating a new one.*

### 2. GitHub Discussions
For general questions, brainstorming new features, or sharing how you use OpenCLI in your daily workflow, head over to GitHub Discussions (if enabled on the repository).
- [Join the Discussion](../../discussions)

### 3. Documentation
Our `README.md` acts as the primary documentation source. It covers installation, usage, configuration, and advanced features like the Model Context Protocol (MCP) and Sandboxing.

## Troubleshooting

Before opening an issue, consider checking:
1. Ensure you are on the latest version of OpenCLI (`npm update -g @zjshen/opencli`).
2. Verify that your API keys (`GEMINI_API_KEY` or `ANTHROPIC_API_KEY`) are correctly set in your environment or `.env` file.
3. If a tool fails due to permissions, ensure you aren't trying to modify files outside the current working directory without explicit user approval.
